# Password-Based Door Lock System

## Overview
A password-based secure door lock system developed using Arduino for access control and authentication.

## Components Used
- Arduino Uno
- Keypad
- LCD Display
- Servo Motor
- Buzzer

## Features
- Password authentication
- Secure access control
- Servo-based door locking mechanism
- LCD status display
- Buzzer alert for incorrect password

## Working
The system accepts password input through a keypad. The entered password is validated using Arduino programming. If the password is correct, the servo motor unlocks the door; otherwise, the buzzer alerts unauthorized access.
